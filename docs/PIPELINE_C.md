# Pipeline C — Standalone Experiments

These are independent scripts that require `Data_ST/` (a static dataset directory
that must be prepared separately). They are **not** part of the main automated pipeline
(`07_master_run.py`) but serve as supplementary comparisons.

---

## 05_final_comparison.py — Baseline vs Combined (TDA+SD)

**Purpose:** Compare baseline against a dataset that combines both TDA and SD augmentations.

**Requirement:** `Data_ST/` directory must contain:
```
Data_ST/
├── baseline/train/
├── tda_x5/train/
├── sd_x5/train/
└── test/       ← created by this script from Data_OG/
```

```bash
cd leaf-disease-ai
python tomato_vs/05_final_comparison.py
python tomato_vs/05_final_comparison.py --output_dir tomato_vs/Results/final_cmp
```

**Output:** `Results/{timestamp}_final_comparison/metrics_summary.csv`, confusion matrices

---

## 06_transfer_learning_comparison.py — Pretrained vs From-Scratch

**Purpose:** Compare EfficientNet-B0 with ImageNet pre-training vs random initialization.

**Requirement:** Same `Data_ST/` directory as above.

```bash
cd leaf-disease-ai
python tomato_vs/06_transfer_learning_comparison.py
```

**Output:** Two result directories:
- `Results/{timestamp}_noptr_nofre/` — No pretrained, no freezing
- `Results/{timestamp}_ptr_nofre/` — Pretrained, no freezing

---

## Note on Data_ST

`Data_ST/` is created externally (e.g., from a previous pipeline run) and contains
fixed training datasets used as a stable baseline for the standalone comparisons.
It is **not** automatically generated by `07_master_run.py`.
